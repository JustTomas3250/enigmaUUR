# Enigma Emulator

An interactive simulator of the World War II Enigma cipher machine, built with React.

## Requirements

- Node.js version 18 or higher
- npm (included with Node.js)

## Installation

```bash
npm install
```

## Running the app

```bash
npm run dev
```

The app will be available at `http://localhost:5173`.
